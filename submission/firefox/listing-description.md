# Respectful Reaction Button

Respectful Reaction Button is an unofficial draft-only helper for linkedin.com.

It adds a clearly labeled `Dislike Draft` action on supported posts, comments, and article pages.
When clicked, the add-on opens LinkedIn's native composer and inserts a respectful draft locally
for the user to review before posting.

What it does:

- adds a draft-only action on supported linkedin.com surfaces
- opens LinkedIn's native composer or reply box
- inserts a respectful draft locally for review
- lets the user enable or disable posts, comments, and articles independently from the popup

What it does not do:

- it does not auto-submit content
- it does not transmit page data outside the browser
- it does not collect analytics or use remote configuration
- it does not run outside `https://www.linkedin.com/*`
