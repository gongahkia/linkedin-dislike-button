Respectful Reaction Button

A draft-only Safari extension for linkedin.com. It opens LinkedIn's native composer, inserts a respectful draft locally, and never posts automatically.

Extension status

You can turn on the extension in Safari extension settings.

The extension is on. You can turn it off from Safari extension settings.

The extension is off. Turn it on from Safari extension settings to use it on linkedin.com.

What it does

Adds a clearly labeled Dislike Draft button on supported posts, comments, and article pages so the user can draft a respectful reply for review.

What it does not do

It does not auto-submit content, collect analytics, transmit LinkedIn page data, or run anywhere outside linkedin.com.
